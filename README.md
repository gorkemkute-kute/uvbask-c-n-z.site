# UVBASKICINIZ Kurumsal Site

Bu paket GitHub Pages, Netlify, Vercel veya cPanel üzerinde çalışabilen statik HTML/CSS/JS sitedir.

## Dosya yapısı

- `index.html` → site içeriği
- `style.css` → renkler, tema, mobil uyum ve tasarım
- `script.js` → koyu/açık mod, mobil menü ve animasyonlar
- `assets/logo-uvbaskiciniz.png` → site logosu
- `assets/favicon.png` → tarayıcı ikonu
- `assets/og-preview.jpg` → sosyal medya paylaşım görseli

## GitHub Pages güncelleme

1. Zip dosyasını bilgisayarda çıkarın.
2. GitHub reposuna girin.
3. `Add file > Upload files` deyin.
4. Zip içindeki dosyaları ve `assets` klasörünü sürükleyip bırakın.
5. `Commit changes` butonuna basın.
6. Site birkaç dakika içinde otomatik güncellenir.
